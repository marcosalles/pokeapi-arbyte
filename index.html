    PokeArbyte é um projeto de ensino da Arbyte

    O objetivo dessa página é ensinar conceitos de HTML, CSS e Javascript através do uso de um framework amplamente difundido, o Materialize.

    Todos os dados utilizados nesse site vêm da API pública PokéAPI, um projeto muito querido realizado por Paul Hallet e Zane Adickes. Thank you very much. A comunicação com a API é realizada pelo AXIOS.

    Essa página foi construída durante uma live com @marcosalles e @rafaelmbsouza, ambos tutores apaixonados da Arbyte. Você pode ver o vídeo completo da live no link nossa Navbar.

    A arbyte é uma escola profissionalizante, que ensina o que é necessário para entrar no mercado de trabalho. Nossos cursos duram até seis meses, e você só paga quando estiver empregado.
